Learn From the Cat the Image Roles in a Web Page

The Cat says that an image can fall in one of the three categories below:

  1. Decorative Images
  2. Informative Images
  3. Images Containing Text

Now let's describe below what the Cat taught me about any of these image types:


Decorative Image

is an image that has purely esthetic role.
Thus, such an image does not transmit any information.

For example, consider the image in the top left of the page, containing the silhouette of a walking cat: it doesn't transmit any information to the user, but it improves the look of the page. Thus, even if this picture also needs to have an "alt" attribute, its value has to be void.
It's worth mention that this image is placed inside the <h1> element and this is perfectly legit. In this situation the fact that its "alt" attribute is void, helps in the case of people who use screen readers or command line browsers, because otherwise the description of the picture would interfere with the <h1> element content in their case.


Informative Image

is an image whose main role is to transmit information to the user.

These pictures are considered Informative Images because they teach you how a Tabby Siberian Cat looks like as a kitten and as an adult.

From what we've learned above from the Cat, we can say that the Decorative Images only role is about Presentation, the main role of the Informative Images is about Content and an Image Containing Text is usually dealing with both roles.
